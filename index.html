CSCI.712 - Computer Animation: Algorithms and Techniques

Projects created in Godot showcasing computer animation from a programmer's perspective. Approaches that will be explored include keyframing systems; kinematics, motion of articulated figures, procedural and behavioral systems, and the use of motion capture data.

Animation Framework

Used Godot to create the framework and testbed for the animation techniques to be explored during the semester. Teapot model from CGTrader.

Keyframing

Implemented a keyframing system to animate a 3D model along a spline path. The model's position and orientation are interpolated between keyframes using linear interpolation for position and spherical linear interpolation (slerp) for rotation.

  • Reads keyframe data from a user-inputted txt file.
  • Uses a Catmull-Rom curve for smooth translation interpolation between keyframes.
  • Custom Easing Graph to control the speed of the animation along the spline.
  • Implements Casteljau Construction of position and rotation variables for efficient evaluation.
